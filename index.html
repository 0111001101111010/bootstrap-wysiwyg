bootstrap-wyswyg very small wyswyg for boostrap

About

Tiny (5K) JQuery Bootstrap plugin to turn any DIV into a WYSIWYG rich-content editor, inspired by CLEditor and bootstrap-wysihtml5. Here are the key features:

  • Bootstrap compatible and consistent with declarative data configuration
  • Automatically binds standard hotkeys
  • Uses standard browser features, no magic
  • Does not create a separate frame, backup text areas etc - instead keeps it simple and runs everything inline in a DIV
  • Requires a modern browser (HTML5 contenteditable)

Usage

$('#editor').wyswyg();
