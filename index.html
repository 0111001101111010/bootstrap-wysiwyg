About

This tiny (2KB) JQuery Bootstrap plugin turns any DIV into a WYSIWYG rich-content editor, inspired by CLEditor and bootstrap-wysihtml5. Here are the key features:

  • Bootstrap compatible and consistent with declarative data configuration
  • Allows a custom built toolbar, no magic markup generators, enabling the web site to use all the goodness of Bootstrap, FortAwesome and so on...
  • Uses standard browser features, no magic non-standard code, toolbar and keyboard configurable to execute any supported browser command
  • Automatically binds standard hotkeys for common operations on Mac and Windows
  • Does not force any styling - it's all up to you
  • (Optionally) cleans up trailing whitespace and empty divs and spans to prevent fake updates
  • Does not create a separate frame, backup text areas etc - instead keeps it simple and runs everything inline in a DIV
  • Requires a modern browser (tested in Chrome 26, Firefox 19, Safari 6)

Usage

$('#editor').wysiwyg();

Don't forget to style your editor div:

 #editor {overflow:scroll; max-height:300px}

Optionally, also create a toolbar (see the source of this page for an example):

<div class="btn-toolbar" data-role="editor-toolbar"
        data-target="#editor">
  <a data-edit="...">...</a>
  ...
</div>

Use standard JQuery methods to access and set content and focus. You can also ask for cleaned up HTML content:

 $('#editor').cleanHtml()

Why?


While building a content editor for MindMup we found plenty of good HTML5 WYSWYG editors online, but most were duplicating functionality that already exists in JQuery and Bootstrap, implementing things that were meanwhile added to HTML5 and supported in major browsers, or doing too much magic such as inserting IFrames with backup text-areas. Most of this was to work around quirks in older browsers (which we didn't need) and caused focus problems on touch devices (which was an issue for us). Too much magic caused problems with bootstrap modals (a must for us). Most editors also build their own toolbars, which embed additional CSS and images and always turn out to be much worse than standard Bootstrap buttons. So we built this tiny tiny editor that does everything we need, does not impose any particular additional CSS and uses modern browser functionality without reimplementing JQuery and Bootstrap.
